# Rangolitico

## Sobre

Entre na torre da comidomancia, resolva quebra-cabeças enquanto busca a Santa ceia, e a receita lendária, jogue sozinho ou com um amigo! Cooperação é essencial para enfrentar os desafios que os aguardam nesta misteriosa fortaleza, onde qualquer pedacinho de comida tem vida e vontade própria. Enfrente torretas de pitaya que cospem fogo, besouros-cebola e diversas outras monstruosidades deliciosas nesta experiência multijogador coop para até dois jogadores!
